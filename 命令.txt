创建git仓库：git init
添加文件：git add [filename]
提交文件：git commit [filename] 输入提交信息后按 :wq 退出
			可以带 -m [提交信息] 一行命令完成带提交信息的提交动作
查看仓库状态：git status   -s(结果简单输出)
查看日志：git log [filename]
删除文件：git rm [filename]

生成非对称公密钥对：ssh-keygen -t rsa

往远程仓库推送代码：
	git remote add [别名] git@github.com:xyongbing/x2.git
	git push -u [别名] master


删除分支
git branch -d [分支名]
